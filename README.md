# Job Management

#### A website used to keep track and log Jobs to do in regards with maintenance

## Requirements

- Bun v 1.x OR npm 10.x
- Node v 20.x

## Installation

### Using Bun:

Install the Required Modules

```bash
  bun install
```

Build the project

```bash
  bun run build
```

Start the Project

```bash
  bun run start
```

### Using NPM:

```bash
  npm install
```

Build the project

```bash
  npm run build
```

Start the Project

```bash
  npm run start
```

### Viewing the site:

Visit http://localhost:4000/ in your browser
